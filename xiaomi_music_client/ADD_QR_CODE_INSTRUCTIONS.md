# 添加赞赏码图片说明

## 📱 如何添加您的微信赞赏码

### 步骤1：保存图片
将您发给我的赞赏码图片保存到电脑上，重命名为：`sponsor_qr_code.png`

### 步骤2：放置图片
将图片文件复制到以下位置：
```
xiaomi_music_client/assets/images/sponsor_qr_code.png
```

### 步骤3：替换现有文件
删除或替换现有的空白 `sponsor_qr_code.png` 文件

### 步骤4：重新构建应用
```bash
flutter clean
flutter pub get
./build_android_obfuscated.sh
```
